麻黃
=========================


麻黃
性味辛、微苦，溫
歸經肺膀胱
主治發汗解表，宣肺平喘，利水消腫
運用1.用於風寒表實證 2.用於咳喘實證 3.用於風水水腫
藥性    




● 麻黃是治喘之要藥

● 麻黃是發汗之第一要藥

● 麻黃暢造成心悸，在使用上要注意心臟不好者禁用

● 石膏是中風傷寒的用藥，得麻、桂而助青龍(麻黃)之力; 得知母、甘草有白虎之號 

● 石膏與麻黃同用就不會大汗出而衰竭

● 麻黃配桂枝則發汗，配石膏則行水，配射干則定喘

● 「太陽無汗代麻黃湯」: 蒼朮三錢，防風二錢，炙甘草用一錢(神朮散); 易蒼朮為白朮-->代桂枝湯(白朮散) 

● 還魂湯其實就是麻黃湯，只是說麻黃湯的桂枝要改成肉桂

● 四季咳嗽用藥建議:
  • 春季: 旋覆花、款冬花
  • 夏天:麥冬、五味子、人參
  • 秋天: 麻黃、黃芩
  • 冬天: 麻黃根、乾薑


● 麻黃發汗，麻黃根止汗

● 辛溫解表、宣肺利尿之要藥：麻黃

● 「夏月麻黃」：香薷

麻黃科植物: 麻黃+麻黃根

             
 生附子、麻黃的比較
-同異
生附子強心陽溫心陽，心臟搏動力量加強(搏動速度不會加快)
麻黃發陽，會使心臟搏動的速度加快
 


【相關藥對】:

    ● 麻黃+桂枝 : 治療四肢水腫，外感風寒表實證。

    ● 麻黃+杏仁 : 治療咳喘。

    ● 麻黃+石膏 : 治療邪熱壅肺的咳喘，全身水腫。

    ● 麻黃+熟地黃 : 治療肺腎陰虛咳喘。

    ● 麻黃+炮附子 : 治療陽虛外感或風寒痹痛。

    ● 麻黃+白朮 : 治療水腫初起或風濕痹證。

    ● 麻黃+薏苡仁 : 治療風濕身疼痛。

    ● 麻黃+硫黃 : 治療水腫，全身水腫。

    ● 麻黃+細辛 : 治療頭痛，四肢疼痛，腰痛，鼻流清涕，咳嗽痰清稀。

    ● 麻黃+蒼朮 : 治療全身水腫，顏面水腫如光鏡者。

    ● 麻黃+炮附子+細辛 : 治療：1.素體陽虛，外感風寒證。

    ● 麻黃根+浮小麥 : 1.體虛多汗，自汗諸症；2.陰虛有熱，盜汗等症。

    ● 射干+麻黃 : 1.痰涎壅盛，氣道不得宣暢，以致氣逆而喘、喉中痰阻、如水雞聲樣痰鳴等症；2.慢性氣管炎，支氣管哮喘，偏於寒者可用。

    ● 麻黃+浮萍 : 1.水腫為病，症見發病急驟、發熱惡風、面目四肢浮腫、骨節疼痛、小便不利者；2.急性腎炎，表現為腰以上腫甚，而兼見表症者；3.感受風寒，風疹瘙癢等症。

桂枝
=========================


桂枝
性味辛、甘，溫
歸經肺心膀胱
主治發汗解肌，溫經通脈，通陽化氣
運用1.用於外感風寒表證 2.用於寒凝血滯的痹證，脘腹冷痛，痛經，經閉等症 3.用於胸痹，痰飲，水腫及心動悸，脈結代
藥性    




● 桂枝治療甚廣，是經方的要藥!

● 桂枝走四肢，在治療四肢問題時是引經藥。

● 桂枝常用來治風寒濕痛。

● 桂枝是治胸悶痛、氣上衝的要藥。

● 桂枝的一字聯想: 行

● 桂枝湯證而有渴的時候就加栝蔞根，因腸胃的津液不夠而發炎<--栝蔞根能夠消炎

● 麻黃配桂枝則發汗，配石膏則行水，配射干則定喘

● 「太陽無汗代麻黃湯」: 蒼朮三錢，防風二錢，炙甘草用一錢(神朮散); 易蒼朮為白朮-->代桂枝湯(白朮散) 

● 還魂湯其實就是麻黃湯，只是說麻黃湯的桂枝要改成肉桂

● 桂枝是用在心陽虛，肉桂用在腎陽虛

● 指（趾）麻的單味藥使用:  氣虛，藥用黃耆；血虛，藥用白芍；寒，用桂枝、附子

● 桂枝辛散，芍藥酸收，一陰一陽，調和營衛。

樟科植物: 桂枝+肉桂+蓽澄茄+烏藥+樟腦


【相關藥對】:

    ● 麻黃+桂枝 : 治療四肢水腫，外感風寒表實證。

    ● 桂枝+芍藥 : 治療外感風寒表虛證。

    ● 桂枝+炙甘草 : 治療心陽虛之心悸氣短，其人欲兩手交叉覆蓋，喜按心胸部位。

    ● 桂枝+吳茱萸 : 治療衝任虛寒，少腹痛，月經痛。

    ● 牡丹皮+桂枝 : 治療血瘀之經閉、痛經。

    ● 茯苓+桂枝+白朮+炙甘草 : 治療中陽不足之痰飲。胸脇支滿，目眩心悸，短氣而咳，舌苔白滑，脈弦滑或沉緊。

    ● 羌活+川芎+桂枝 : 治療後頭痛。

    ● 桂枝+白芍+當歸 : 治療左肩膀僵硬。

    ● 桂枝+白芍+黃耆 : 治療右肩膀僵硬。

    ● 葛根+炮附子+桂枝+白芍+羌活 : 治療膏肓痛、背痛。

    ● 桂枝+蒼朮 : 治療風濕痛，退化性關節炎等。

    ● 大黃+桂枝 : 治療下腹拘急硬痛、小便自利、夜晚發熱，譫語煩渴、甚則如狂，以及血瘀經閉，痛經，產後惡露不下，脈沈實或澀。

    ● 牡丹皮+川芎+赤芍+桂枝+炮薑 : 治療血瘀證。經痛，閉經。

    ● 桂枝+炮附子+蒼朮 : 治療全身痺痛。

    ● 桂枝+炮附子 : 治療寒凝血滯的痹證。全身疼痛，或脘腹冷痛，或經痛、閉經。

    ● 枳實+薤白+桂枝 : 治療胸痹，心中痞氣，氣結在胸，胸滿，脅下逆搶心。

    ● 桂枝+豬苓+澤瀉 : 治療眩暈，口渴，小便不利。

    ● 茯苓+桂枝+白朮+炙甘草+薑半夏 : 治療眩暈證，小便不利，舌苔白膩而滑。

    ● 桂枝+蒼朮+薏苡仁 : 治療風濕，全身疼痛，身體沈重，苔白膩。

    ● 醋鱉甲+鬱金+黨參+桂枝 : 治療慢性肝炎。

    ● 桂枝+白芍+葛根 : 治療肩背痛。

    ● 桑枝+桂枝 : 治療中風、半身不遂，或上肢麻痺。

    ● 桂枝+龍骨+牡蠣 : 治療氣上衝，失眠，肝陽上亢。

    ● 桂枝+牡蠣 : 1.痛風病；2.胃、十二指腸潰瘍，證屬虛寒型者。

細辛
=========================


細辛
性味辛，溫；有小毒
歸經肺腎心
主治祛風解表，散寒止痛，溫肺化飲，通竅
運用1、用於外感風寒及陽虛外感證2、用於頭痛，痹痛，牙痛等痛證3、用於寒飲咳喘

● 細辛之治症必有水泛(濕)。

● 細辛可在有寒濕時用於止痛

● 細辛的一字聯想: 水

● 治寒飲伏肺之要藥:細辛

● 鼻淵及其引起之頭痛: 辛夷 蒼耳子 細辛

馬兜鈴科植物: 細辛+尋骨風+關木通+青木香+天仙籐+馬兜鈴


【相關藥對】:

    ● 細辛+白芷 : 治療牙痛。

    ● 麻黃+細辛 : 治療頭痛，四肢疼痛，腰痛，鼻流清涕，咳嗽痰清稀。

    ● 熟地黃+細辛 : 腰痛

    ● 細辛+烏藥 : 1.下腹諸痛證

    ● 細辛+炮附子 : 腳冷、腳麻、頻尿等。

    ● 細辛 : 1、用於外感風寒及陽虛外感證2、用於頭痛，痹痛，牙痛等痛證3、用於寒飲咳喘

生薑
=========================


生薑
性味辛，溫
歸經肺脾
主治發汗解表、溫中止嘔，溫肺止咳
運用1、用於外感風寒表證2、用於多種嘔吐證3、用於風寒咳嗽
藥性     




● 生薑的一字聯想: 逆

● 四季料理用薑:   春天å薑，夏天生薑，秋天老薑，冬天乾薑

● 生薑作用在胃，炮薑作用在小腸，乾薑作用在大腸(肺)

● 治療尿毒需發汗，用大量的”生薑”

● 生薑用來散胃中的水，半夏用來散胃外面的水

● 皮膚白斑(白癜風):  白附子跟補骨脂用泡酒、泡水、煮水皆可，用生薑沾之來做局部塗抹。 二者可以刺激黑色素的分泌。內服沒有效果! (補骨脂酊是用25%的酒精600CC，來配白附子一兩，補骨脂一兩，和生薑五錢)

● 半夏、天南星、芋頭這類天南星科植物都含有豐富的生物鹼，接觸到皮膚就會產生搔癢，甚至引起紅腫熱痛的反應。解決之道:  生薑第一，其次是鹽巴。 

● 「嘔家聖藥」: 生薑

● 魚蟹中毒: 紫蘇,生薑,

● 胃寒嘔吐：丁香 生薑 吳茱萸 高良姜  沉香

             
 生薑、乾薑的比較
-同異
乾薑散寒作用在肺，質輕在上。偏溫肺化飲。去陰實之寒
生薑作用在胃，質重在下。偏溫中散寒。去陽虛之寒
 


【相關藥對】:

    ● 生薑+薑半夏 : 治療寒飲嘔吐，失眠，容易焦躁緊張、心驚。

    ● 生薑+大棗 : 治療風寒感冒(入表藥)，胃脘不舒嘔吐(入健脾藥)。

    ● 薑半夏+厚朴+茯苓+生薑+蘇子 : 治療咳喘，一直有痰_白色黏痰，痰白或清晰，苔白膩而滑。

    ● 薑半夏+厚朴+茯苓+生薑+紫蘇葉 : 治療咳喘，一直有痰_白色黏痰，痰白或清晰，苔白膩而滑。

    ● 生薑+竹茹 : 治療嘔吐，噁心想吐。

    ● 吳茱萸+炮薑+肉桂+生薑 : 治療腹部寒證，腹痛，喜溫喜按，舌淡苔白，脈弦緊。

    ● 吳茱萸+生薑 : 治療：1.寒凝肝脈諸痛，如頭頂痛。

    ● 大黃+生薑+生半夏+茯苓 : 治療腎衰時的噁心嘔吐。

    ● 生薑+薑半夏+茯苓 : 寒飲嘔吐。

香薷
=========================


香薷
性味辛，微溫
歸經肺胃脾
主治發汗解表，化濕和中，利水消腫

藥性

[top]   [herb]一六三、麻黃：

麻黃【本經原文】味甘，溫，無毒。治中風，傷寒，頭痛，溫瘧，發表出汗，去邪熱氣，止欬逆上氣，除寒熱，玻癥堅積聚。一名龍沙。
麻黃【產地】我國西北部皆有出產，以山西大同為最良，其他如印度及西伯利亞南非洲皆有產。
麻黃【性味】味苦，性溫散，無毒。
麻黃【主治】麻黃為發汗解散要藥，功能退熱平喘利水，主傷寒表實無汗，治肺實喘逆，通利水道。
麻黃【別錄】五臟邪氣緩急，風脅痛，止好睡，通腠理，解肌，洩邪惡氣，消赤黑斑毒，不可多服，令人虛。
麻黃【甄權】治身上毒風疹痺，皮肉不仁，主壯熱溫疫，山嵐瘴氣等。
麻黃【大明】通九竅，調血脈，開毛孔皮膚。
麻黃【元素】去營中寒邪，洩衛中風熱。
麻黃【綱目】散赤目腫痛，水腫風腫，產後血滯。
麻黃【容川】雖其莖直上，但卻如草之叢生，故能上升又能外散，其莖空，直達而上，且無大味，純得天輕陽之氣，故專主氣分。一般取其苗，以其苗細長中空，象人毛孔，而氣輕揚，故能發汗，能通陽氣，直走鼻皮毛，其根堅實味澀，故能止汗。
麻黃【炮製】揀去雜質，去盡木質及殘根，嘖淋清水，微潤，切1.5厘米段片，曬乾或烘乾，去灰即得。
麻黃【藥徵】主治喘咳水氣也，旁治惡風惡寒，無汗，身疼，骨節痛，一身黃腫。
麻黃【靈胎】發汗止喘。
麻黃【用量】普通五分至一錢，大劑三錢。
麻黃【禁忌】平日陽虛，腠理不密之人忌之，諸虛有汗，肺虛痰嗽，氣虛發喘，均忌。
麻黃【倪註】麻黃為發汗第一要藥，能退熱平喘利水，皆其解散之功，凡風寒初襲肌表，不得解散則發為頭痛項強惡寒諸症狀，發其汗則風寒得以解散，諸症自除，舉凡外感初起之發熱，多由汗不得出，因勢利導使汗出而熱解，比本經之所言去邪熱也，又肺受風寒不解，則咳氣上逆，用麻黃利水道，外為發汗，是故痰水因而下行，風寒因而得以宣洩，因病根解除，咳氣上逆自癒，若以之點眼，可使瞳孔放大，此因其能散。今麻黃配桂枝能散營分之寒邪，配膏能洩衛分之風熱，皮肉不仁，著名方劑如麻黃湯，麻杏甘石湯，大青龍湯，應麻黃根能止汗，節亦止汗，故欲發汗，則須去根與節。

[top]   [herb]九十二、牡桂：(桂枝)

牡桂【本經原文】味辛溫，主上氣欬逆，結氣，喉痹，吐吸，利關節，補中益氣，久服通神輕身不老。一名肉桂，一名桂枝，一名桂心。桂之供藥有兩種，一曰菌桂，又稱小桂，其皮為肉桂，或稱筒桂。二曰牡桂，又稱大桂，或單稱桂，其皮為官桂，其枝為桂枝。
牡桂【產地】產安南及我國雲南兩廣等地。
牡桂【性味】味辛甘，性溫無毒。
牡桂【主治】桂枝為發表解肌要藥，主溫經通脈，表虛中風，治手足痛風，傷寒頭痛，無汗能發，有汗能止，行肺氣，散寒邪，調和營衛，橫通肢節。
牡桂【別錄】心痛脅痛脅風，溫經通脈，止煩出汗。
牡桂【甄權】去冷風疼痛。
牡桂【丹溪】橫行手臂治痛風。
牡桂【元素】去傷風頭痛，開肌理，解表發汗，去皮膚風溼。
牡桂【容川】色赤味辛，入心肝助火，但桂枝實因苓澤之利水，方能入水以化氣，可以導心火以下交於水，故多用之以降衝逆之氣，加入補血藥中，因而能溫通血脈，桂枝能散四肢，質較堅實，故桂枝能兼走筋骨。味辛又得土之氣，故能散血脈肌肉中之風寒。
牡桂【藥徵】主治衝逆也，旁治奔豚頭痛，發熱惡風，汗出身痛。
牡桂【用量】普通三分至錢半。
牡桂【禁忌】凡陰虛火炎，溫病大熱，喉症血症均忌。

紫蘇【產地】處處有之，為一年生之草本，葉之背面皆紫者佳，有香氣，莖葉與子，皆入藥。
紫蘇【性味】味辛，性溫，無毒。
紫蘇【主治】紫蘇為發表散寒解毒要藥，功能降氣化痰，發汗袪寒，主傷風寒熱霍亂，肺氣喘急，心腹脹滿，安胎婦人胎前宜之。
紫蘇【別錄】主下氣，除寒熱，其子尤良。
紫蘇【孟銑】除寒熱，治一切冷氣，
紫蘇【日華】補中益氣，治心腹脹滿，止霍亂轉筋，開胃下食，止腳氣通大小腸，
紫蘇【蘇頌】通心經，益脾胃，煮飲尤勝，與橘皮相宜。
紫蘇【蘇恭】下氣消緩，虛者宜之，主安胎。
紫蘇【用量】葉梗普通八分至二錢，蘇子錢半至三錢。
紫蘇【禁忌】凡陰虛因發寒熱，或寒頭痛者，宜斂宜補，不可用蘇葉，火升作嘔者亦忌，惟可用子，忌鯉魚，同食發惡瘡。
紫蘇【容川】紫蘇色紅味辛香，能散血分肌肉中之寒，枝葉披離，故主散之性多，而主升之性少。
紫蘇【備註】蘇子：主上氣欬逆冷氣，及腰腳中濕氣，風結氣，調中止霍亂，消五臟嘔吐反胃，消痰止嗽，利大小便，治肺氣喘急。蘇葉：主發散，孟銑曰：『蘇子與蘇葉同功，發散風氣宜用葉，清利上下宜用子。』近世多用以發散風寒，至化痰之力，則不如子也。主散氣分之熱，如喉為氣之海。蘇梗：主順氣表散。汪昂曰：『葉發汗散寒 梗順氣安胎，子開鬱降氣，消痰定喘。』

防風【本經原文】味甘溫無毒，主大風頭眩痛，惡風風邪，目肓無所見，風行周身，骨節疼痺煩滿，久服輕身。
防風【產地】產蒙古及山西河南等地，根黃白色，入藥。
防風【性味】味甘辛，性溫，無毒。
防風【主治】防風為傷風頭痛要藥，功能散表發汗，主諸般風，周身痛，搜肝瀉肺，散風勝溼。
防風【別錄】煩滿脅痛，風頭面去來，四肢攣急，字乳，金瘡內痙。
防風【大明】治三十六般風，男子一切勞累，補中益神，風赤眼，止冷淚，及癱瘓，通利五臟關節，五勞七傷，羸損盜汗，心煩體重，能安神定志勻氣脈。
防風【元素】治上焦風邪，瀉肺實，散頭目中滯氣，經絡中留溼，主上部見血。
防風【克潛】防風善表散，故用以解毒，如砒毒，鳥頭毒，芫花毒，野菌毒，諸毒，又防風本制黃耆，黃耆得防風而功最大，治風邪久留不去，表虛自汗不止。徐之才曰：『得蔥白行周身，得澤瀉稾本療風，得歸芍陽起石禹餘糧治婦人子臟風。』元素曰：『治風去溼之仙藥也。』
防風【容川】防風辛而味甘入肌，氣香而溫，故入脾，散肌肉之風寒。
防風【靈胎】專治風邪。
防風【用量】普通一錢至三錢。
防風【禁忌】似中風，產後血虛發痙，諸病血虛痙急，頭痛不因風寒，溏瀉不因寒溼，二便秘澀，小兒慢驚，陰虛盜汗，陽虛自汗，均忌。

辛夷【本經原文】味辛溫，主五藏身體寒風，頭腦痛，面皯，久服下氣，輕身明目，增年耐老。
辛夷【產地】處處有之，庭園可以種植。樹高大，苞入藥。
辛夷【性味】味辛，性溫，無毒，
辛夷【主治】辛夷為升陽治上要藥，主頭腦各病，鼻淵鼻塞，行氣，宣肺氣。
辛夷【別錄】溫中解肌，利九竅，通鼻塞涕出，治面腫，引齒痛，眩冒身几几如在車船之上者
辛夷【大明】通關脈，治頭痛，憎寒體噤，搔癢，入面脂生光澤。
辛夷【靈胎】疏肝散風。
辛夷【用量】普通一二錢。
辛夷【禁忌】凡氣虛人忌，偶感風寒鼻塞亦忌，頭腦痛屬血虛火熾亦忌，齒痛屬胃火者亦忌，惡赤石脂，畏菖蒲黃連蒲黃石膏。
辛夷【備註】此藥性升，故主頭面各疾，可引諸藥上升至頭面，經曰腦滲為涕，王冰曰腔膽不澄，則為濁涕，如泉不已，故名鼻淵。
辛夷【容川】辛夷花在樹梢，其性極升，而味辛散，故能散腦鼻間風寒。
辛夷【炮製】揀去雜質及柄，去灰即得。
