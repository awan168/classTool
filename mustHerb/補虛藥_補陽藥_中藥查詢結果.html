鹿茸
=========================


鹿茸-鹿角膠-鹿角霜
性味甘、鹹，溫
歸經腎肝
主治壯腎陽，益精血，強筋骨，調沖任，固帶脈，托瘡毒
運用1、用於腎陽不足，精血虧虛的陽痿早泄，宮寒不孕，尿頻不禁，頭暈耳鳴，腰膝酸痛，肢冷神疲等證2、用於肝腎不足的筋骨痿軟，小兒發育不良，囟門過期不合，齒遲，行遲等3、用於沖任虛寒，帶脈不固的崩漏不止，帶下過多4、用於瘡瘍久潰不斂，膿出清稀，或陰疽內陷不起
藥性     




● 鹿角帶血者是鹿茸

● 鹿茸是性荷爾蒙刺激素，可以強精蟲。吃素食的陽起石，陽起石是礦物質。鹿茸朝天，善通督脈。

● 溫補腎陽，補督脈、益精血之要藥：鹿茸

    ● 淫羊藿+紫石英 : 1.陽虛宮寒之痛經、閉經、不孕

仙茅
=========================


仙茅
性味辛，熱；有毒
歸經腎肝脾
主治溫腎壯陽，強筋骨，祛風濕，溫脾止瀉

藥性     




● 仙茅需用鹽炒或酒炒。因為仙茅有毒，如果未經炮製，吃下去超過三錢就會中毒，舌頭會腫起來。酒炒仙茅對於脊椎骨跟腎臟有幫助，可溫腎壯陽，強筋骨，祛風濕。

石蒜科植物: 仙茅

補骨脂
=========================


補骨脂
性味辛、苦，溫
歸經腎脾
主治補腎助陽，固精縮尿，暖脾止瀉，納氣平喘
運用1、用於腎陽不足，命門火衰之腰膝冷痛，陽痿，遺精，尿頻等證2、用於脾腎陽虛泄瀉3、用於腎不納氣的虛喘
藥性   




● 補骨脂、砂仁: 治胸悶 (刺激延腦的呼吸中樞，增加肺活量)

● 皮膚白斑(白癜風):  白附子跟補骨脂用泡酒、泡水、煮水皆可，用生薑沾之來做局部塗抹。 二者可以刺激黑色素的分泌。內服沒有效果! (補骨脂酊是用25%的酒精600CC，來配白附子一兩，補骨脂一兩，和生薑五錢)

● 補骨脂和牡蠣可以讓骨頭長的很快。補骨脂能夠壯骨髓、補腎(刺激類固醇分泌)、治全牙痛。

● 【倪師】補腎時，用澤瀉(五錢)可使腎臟休息，再用補骨脂等補腎藥補之。

● 補骨脂、砂仁是能刺激延腦的呼吸中樞的藥對，可增加肺活量，同時可治療胸悶

● 骨碎補、補骨脂、續斷像強力膠把骨頭黏在一起，可修補骨頭

● 晨起就很倦怠(精神上的)或中午過後就超級累--→前一晚製造的命門火不夠: 用補骨脂

● 腎不納氣之喘：蛤蚧 補骨脂 沉香 磁石 紫河車

豆科植物: 葛根+淡豆豉+決明子+苦參+苦豆子+山豆根+綠豆+番瀉葉+海桐皮+雞骨草+刀豆+槐花+降香+雞血籐+蘇木+兒茶+皂莢+合歡皮+黃芪+白扁豆+甘草+補骨脂+沙苑子+胡蘆巴


【相關藥對】:

    ● 補骨脂+菟絲子+葛根 : 治療腎精不足引起的閉經，即女性賀爾蒙不足。

    ● 枸杞子+補骨脂+菟絲子+淫羊藿 : 治療腎虛證。腰膝冷痛，陽痿，遺精，尿頻，泄瀉。

    ● 補骨脂+核桃仁 : 1.腎虛之咳喘諸症；2.腎氣不足，以致腰酸、腰痛、陽痿、遺精、小便頻數、遺尿等症；3.神經衰弱，頭昏、失眠、記憶力減退等症；4.妊娠腰痛，狀不可忍。

    ● 補骨脂+蛤蚧 : 1.慢性支氣管炎、肺氣腫，證屬肺腎兩虛，腎不納氣，喘息氣短，動則尤甚，身憊汗出，腰膝酸軟，咳嗽痰多諸症；2.肺虛勞嗽諸症。

    ● 肉豆蔻+補骨脂 : 1.脾腎陽虛，虛冷泄瀉，日久不癒諸症；2.五更泄瀉，腸鳴腹痛、瀉後則安等症；3.產後瀉痢；4.腰痛，證屬腎陽虛衰者；5.水腫，證屬脾腎陽虛，水濕為患者。

益智仁
=========================


益智-益智仁
性味辛，溫
歸經腎脾
主治補腎助陽，固精縮尿，溫脾止瀉，開胃攝唾
運用1、用於腎氣虛寒之遺精滑精，遺尿尿頻等2、用於脾寒泄瀉，腹中冷痛，口多涎唾等
藥性    




● 收斂藥 : 小便太多了，我們可以用益智仁、芡實、覆盆子; 流鼻涕、口水多、流鼻涕用烏梅、益智仁、覆盆子

● 益智仁可以使平滑肌收縮，時常流口水的小孩吃益智仁就不流口水。益智仁可以抑制唾液的分泌和多尿

● 大人的遺精，可等同就是小孩子的遺尿，所以治療小孩子尿床，大多從大人的遺精方向著眼:   
  • 第一組藥是入腦、通腦竅的藥:  遠志、菖蒲
  • 第二組藥就是用一些鎮靜的藥: 龍骨、牡蠣。
  • 第三組藥是用一些安神的藥: 柏子仁、遠志
  • 最後第四組藥用收澀劑: 蓮蕊鬚、五味子、山茱萸、益智仁、覆盆子


● 大腦皮質開竅藥:  遠志，菖蒲，益智仁 (可治多尿)

● 治脾寒瀉痛或多涎唾之要藥：益智仁

姜科植物: 生姜+砂仁+豆蔻+草豆蔻+草果+干姜+高良姜+炮姜+郁金+薑黃+莪術+益智仁


【相關藥對】:

    ● 茯苓+益智仁 : 1.下元虛寒，氣化功能失調，以致小便淋漓不暢、小便渾濁等症；2.脾腎虛寒，泄瀉等症。

    ● 烏藥+益智仁 : 1.下元虛冷，小便頻數等症；2.小兒遺尿；3.中、老年人前列腺肥大諸症。

    ● 益智仁+萆薢 : 1.腎虛小便混濁不清，尿意頻頻，淋漓不暢等症；2.婦人帶下諸症；3.乳糜尿；4.尿酸性關節炎；5.中老年人前列腺肥大諸症。

    ● 益智仁 : 1、用於腎氣虛寒之遺精滑精，遺尿尿頻等2、用於脾寒泄瀉，腹中冷痛，口多涎唾等

菟絲子
性味甘，溫
歸經肝腎脾
主治補腎固精，養肝明目，止瀉，安胎
運用1.用於腎虛腰痛，陽痿遺精，尿頻，帶下等症 2.用於肝腎不足，目失所養而致目昏目暗，視力減退之證 3.用於脾腎虛泄 4.用於肝腎不足的胎動不安

● 菟絲子為補腎益精的要藥，但同時可治陽強不痿，故精不夠，同時不舉，菟絲子就讓它舉起來，但用之過多，反而會不舉。

● 菟絲子使他的腎臟細胞再生(治腎陰虛)，腎臟發炎、尿毒症可用菟絲子。菟絲子同時可使卵巢的卵再生(肝之臟象)。所以菟絲子作用在肝腎。

● 腎元重新再生就是要借重女珍子和菟絲子

● 菟絲子是雌激素，也是黃體素，攝護腺癌主力藥是菟絲子，也是脫髮第一聖藥

● 補腎固精用五子衍宗丸: 菟絲子、覆盆子、枸杞子、五味子、車前子

● 陰陽並補之佳品：山茱萸 菟絲子

  • 杜仲（肝腎虛虧）
  • 桑寄生（肝腎虛虧）
  • 續斷（肝腎虛虧）
  • 菟絲子（腎虛）

杜仲
=========================


杜仲
性味甘，溫
歸經肝腎
主治補肝腎，強筋骨，安胎
運用1.用於肝腎不足的腰膝酸痛、下肢痿軟及陽痿、尿頻等症 2.用於肝腎虧虛、下元虛冷的妊娠下血、胎動不安，或習慣性流產等症
藥性    




● 綠豆要帶殼才有利反作用，凡是有皮的皆利尿(茯苓皮、杜仲皮、五加皮、綠豆殼)

● 婦女產後皮過鬆（妊娠紋）可用杜仲

鹿茸【本經原文】味甘，溫，無毒。治漏下，惡血，寒熱，驚癇，益氣，強志，生齒，不老。角，溫，無毒。治惡瘡，癰腫，逐邪惡氣，留血在陰中。
鹿茸【產地】鹿產山林中，處處有之，我國西北及東三省為多，其茸及角入藥。
鹿茸【性味】味甘鹹，性溫，無毒。
鹿茸【主治】鹿茸為大補真陽要藥，主養血益陽，強筋健骨，生精充髓，治男子虛勞，老人精衰，腰膝無力，滑精，眩暈，婦人崩漏帶下。
鹿茸【別錄】療虛勞洒洒如瘧，羸瘦，四肢酸疼，腰脊痛，小便數利，洩經溺血，破瘀血在腹，散石淋，癰腫，骨中熱，舉疽癢，安胎下氣，殺鬼精物，久服耐老，不可近丈夫陰令痿。
鹿茸【日華】補男子腰腎虛冷，腳膝無力，夜夢鬼交，精溢自出，女人崩中瘺血，赤白帶下，炙末空心酒服方寸匕，壯筋骨。
鹿茸【容川】其宿以頭顧尾，能通督脈，督者腎脈，坎中一陽之主脈也，鹿生北方，得坎中一陽之氣也，故其督脈極旺，而脊與腦髓極足，因而上發而生角，每年一換，初生為鹿茸，茸之精氣極足，為補髓強精壯陽益血之聖藥。但其性上行，凡是血逆火逆者不宜用。惟血虛火弱，陽不舉，氣不上者，乃為合宜。
鹿茸【靈胎】補陽益血。
鹿茸【用量】普通七分至錢半。
鹿茸【禁忌】壯年慎用，陰虛火旺者忌之，麻勃為使，畏大黃，不可嗅，有蟲恐入鼻顙，藥不能治，男子陰莖近入，易於痿弱。
鹿茸【附註】鹿胎－渾然元氣所在，歸下焦而不上行，為種子益腎補胞宮之要藥。

陽起石【性味】味鹹，性微溫，無毒。
陽起石【主治】陽起石為補命門壯陽道要藥，功能療陰痿，男女下部虛冷，補不足。
陽起石【別錄】療男子陰頭寒，陰下濕癢，去汗臭，消水腫，今人有子。
陽起石【甄權】補腎氣，精乏腰痛，漆冷溼痺，子宮久冷，冷癥寒瘕，止月水不定，
陽起石【大明】治帶下瘟疫，冷氣，補五勞七傷。
陽起石【好古】補命門火不足。
陽起石【靈胎】助陽益氣。
陽起石【用量】普通水飛用，五分至一錢。
陽起石【禁忌】凡陰虛及陽痿屬於失志，以致火氣閉密，不得發越而然，與崩漏由於火盛，而非虛寒者，均忌，忌羊血，凡使用火炙醋淬七次，研細水飛用，不入湯藥，又石藥冷熱皆有毒（因不易消化），非可久服之物。
陽起石【附註】為雲母石之根，其山冬不積雪，夏則生雲，積陽上升，故凡病人陽氣下陷，陽物不舉者，用以升舉陽氣，此以陽助陽。
陽起石【炮製】揀去雜質，搗碎，置鐵鍋中，煅透，趁熱倒入酒漬淬，取出晾乾，研粉即得。
