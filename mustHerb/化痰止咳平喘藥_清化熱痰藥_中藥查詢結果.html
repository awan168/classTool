桔梗
=========================


桔梗
性味苦、辛，平
歸經肺
主治開宣肺氣，祛痰排膿，利咽
運用1、用於肺氣不宣的咳嗽痰多，胸悶不暢2、用於熱毒壅肺之肺癰3、用於咽喉腫痛，失音
藥性     




● 【倪師】不管是甲狀腺腫瘤、鼻咽癌、淋巴腺腫瘤、扁桃腺腫瘤，只要是喉嚨這一段有硬塊，只有兩味藥在用，就是桔梗和連翹(甚者加巴豆)

● 化膿藥: 桔梗、枳實、連翹

● 桔梗二大功能:  排膿、去寒痰

● 諸藥之舟楫---桔梗: 桔梗載諸心藥久留膈上; 桔梗載這個藥物上行三陽的部分，太陽、少陽、陽明

● 失音：訶子 桔梗 胖大海

● 利咽：薄荷 牛蒡子 板藍根 射干 山豆根 馬勃 玄參 巴豆 牛黃 胖大海 桔梗

桔梗科植物: 半邊蓮+桔梗+黨參+南沙參


【相關藥對】:

    ● 桔梗+枳殼 : 治療胸悶咳嗽等症。

    ● 醋香附+枳殼+桔梗 : 治療胸口脹滿不適。

    ● 桔梗+石膏 : 治療咽喉痛，乾咳無痰或黃稠痰。

    ● 桔梗+薑半夏 : 治療咽喉痛。

    ● 桔梗+石膏+知母 : 治療發熱，口渴，咽喉痛，舌紅，脈洪大。

    ● 桔梗+石膏+薑半夏 : 治療咳嗽，不易咳出的濃痰，黏痰。

    ● 薏苡仁+桔梗+甘草 : 治療膿已成而未破。

    ● 桔梗+枳實 : 治療：1.兼起疼痛的化膿性腫瘍，而氣血凝滯，患部緊張，炎性浸潤甚強，呈現堅硬狀態的各種疾患。

    ● 桔梗+石膏+天花粉+葛根 : 治療口乾、咽乾，舌質紅，脈數。

    ● 升麻+桔梗 : 1.咽喉腫痛（急性咽喉炎）；2.牙齦腫痛，證屬風熱蘊毒所致者；3.肺痛（肺膿瘍）。

    ● 訶子+桔梗+甘草 : 1.音嘶、音啞諸症；2.慢性喉炎，喉頭結節（息肉）等喉部疾患，均可使用。

    ● 紫蘇+桔梗 : 一切氣機不暢，以致胸悶不舒、氣逆等症。

    ● 桔梗+枳殼+薤白+杏仁 : 1.氣機不調，胸膈脹悶，脘脹不適，甚則疼痛，食欲不振，大便不利等症；2.急慢性氣管炎，中焦胸膈滿悶，痰氣不暢者；3.冠心病心絞痛，症見胸悶憋氣者；4.呃逆，證屬氣機不調者；5.功能性失語；6.梅核氣諸症。

    ● 桔梗+杏仁 : 痢疾初起，表現為半痢半糞者。

    ● 黃耆+桔梗+甘草 : 瘡瘍成膿不潰，或潰後久不收口，證屬氣血不足者。

[top]   [herb]二七二、桔梗：

桔梗【本經原文】味辛，微溫有小毒。治胸脅痛如刀刺，腹滿，腸鳴幽幽，驚恐悸氣。生山谷。
桔梗【產地】自生於山中向陽之地，處處皆有，藥似沙參，根亦相似而較白。
桔梗【性味】味苦辛，性微溫，有小毒。
桔梗【主治】桔梗為宣肺袪痰要藥，功能治痰壅氣逆，胸脅氣痛，利咽喉，治肺癰。
桔梗【別錄】利五臟腸胃，補血氣，除寒熱風痹，溫中消穀，療喉咽痛，下蠱毒。
桔梗【甄權】治下痢，破血積氣，消聚痰涎，去肺熱氣促嗽逆，除腹中冷痛，主中惡及小兒驚癇。
桔梗【大明】下一切氣，止霍亂轉筋，心腹脹痛，補五勞，養氣除邪辟溫，破癥瘕肺癰，養血排膿，補內漏及喉痺。
桔梗【元素】利竅，除肺部風熱，清利頭目咽嗌，胸膈滯氣及痛，除鼻塞。
桔梗【藥徵】主治濁唾腫膿也，旁治咽喉痛。
桔梗【用量】普通八分至二錢。
桔梗【禁忌】凡攻補下焦藥中勿入。氣逆上升，不得下降，及邪在下焦，均忌。
