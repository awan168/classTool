鉤藤
=========================


鉤藤
性味甘，微寒
歸經肝心包
主治息風止痙，清熱平肝
運用1、用於肝風內動，驚癇抽搐2、用於頭痛，眩暈
藥性   





【相關藥對】:

    ● 鉤藤+杜仲 : 治療高血壓的頭痛、眩暈、失眠。

    ● 膽南星+鉤藤+全蠍 : 治療癲癇，四肢抽搐。

    ● 羚羊角+鉤藤 : 治療：1.肝風內動，驚癇抽搐。

    ● 秦艽+鉤藤 : 治療眼皮跳，顏面神經痙攣，面肌痙攣。

    ● 鉤藤+薄荷 : 1.風熱感冒，或溫病初起，症見發熱，微惡寒、無汗、頭痛、身痛者；2.內傷、外感咳嗽，且久久不癒者；3.風熱上擾，症見頭昏、頭痛、視物不明者；4.肝陽上擾，以致頭脹頭痛、頭暈目眩等症；5.小兒夜寐不安，驚抖，咳嗽。

    ● 全蠍+鉤藤 : 1.頑固性頭痛、久久不癒者，證屬風熱為患者；2.面神經麻痹（面癱），面肌經痙攣，三叉神經痛（面痛）；高血壓病、動脈硬化所引起的頭痛、頭昏、頭暈等症；4.小兒驚癇。

    ● 鉤藤+天麻 : 1.眩暈，頭痛，四肢麻木、抽掣，證屬肝風內動者；2.頭皮、皮膚瘙癢諸症；3.高血壓病；4.梅尼埃綜合徵。

    ● 鉤藤+川牛膝 : 腦血管痙攣，高血壓病，表現為肝陽上亢、頭暈目眩、頭脹頭痛、半身麻木等症，均宜使用。

    ● 鉤藤+桑寄生 : 1.高血壓病，證屬肝腎不足，肝陽上擾者；2.冠心病心絞痛，證屬肝腎不足者。

    ● 鉤藤+白頭翁 : 1.帕金森病，證屬血熱風動者；2.甲狀腺功能亢進，證屬血熱風動者；3.神經官能症，症見手抖等。

    ● 鉤藤+川芎 : 治頭痛(抽痛、緊痛)。

[herb]蜈蚣【本經原文】味平，溫，有毒。治鬼疰，蠱毒，噉諸蛇、蟲、魚毒，殺鬼物老精，溫瘧，去三蟲。生川谷。

蜈蚣【產地】處處有之，赤足者良。
蜈蚣【性味】味辛，性溫，有毒。
蜈蚣【主治】蜈蚣為驅風散結要藥，主小兒驚癇，風搐臍風，治寒熱積聚，墮胎，去惡血，制蛇毒。
蜈蚣【別錄】療心腹寒熱積聚，墮胎，去惡血。
蜈蚣【日華】治癥癖。
蜈蚣【用量】普通一至二錢。
蜈蚣【禁忌】蜈蚣有毒，必風氣暴烈，藥病相當，乃可設，或過劑，以蚯蚓桑皮解之，畏蛞蝓。被蜈蚣螫傷，搗蛞蝓塗之即止痛。
