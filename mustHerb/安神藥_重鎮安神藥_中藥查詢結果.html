朱砂
=========================


朱砂
性味甘，寒；有毒
歸經心
主治鎮心安神，清熱解毒
運用1、用於心神不寧，心悸，失眠2、用於驚風、癲癇3、用於瘡癰腫毒，咽喉腫痛，口舌生瘡
藥性  




● 雄黃、朱砂類的藥物含汞量比較高，用量宜輕

● 眼科藥很澀，我們加點朱砂下去，然後就變得很滑

● 朱砂可作用在心臟，心律不整、心悸可以用朱砂，又可以作用在大腦而有鎮靜作用。朱砂並可調經、利尿


【相關藥對】:

    ● 朱砂+黃連 : 治療心火亢盛之心悸失眠。

    ● 珍珠母+磁石+朱砂+神曲 : 1.肝腎不足，肝陽上逆，以致頭暈、眼花、瞳孔散大、視物不明，以及耳鳴、耳聾等症；2.高血壓病，症屬虛性者；3.青光眼。

    ● 朱砂+琥珀 : 1.心神不寧，失眠多夢，寐而不實，亂夢紛紜等症；2.中老年陣發性心房纖顫，症見難眠易醒、寐少夢多、恍惚不安等；小兒胎癇，職是婦人難產，由於產程過長，以致胎兒腦缺氧所致。

磁石
=========================


磁石
性味鹹，寒
歸經心肝腎
主治鎮驚安神，平肝潛陽，聰耳明目，納氣定喘
運用1、用於心神不寧，驚悸，癲癇2、用於肝陽眩暈3、用於肝腎虧虛，目暗耳聾4、用於腎虛喘促
藥性 




● 磁石為補腎強陰要藥，取其能引肺氣入腎，使陰陽安宅，鎮驚，明目，通耳，益精

● 凡石藥皆有毒，獨磁石沖和無悍猛之氣，然體重，難於消化，漬酒優於丸散，凡使火炙醋淬，研末水飛，或醋煮三日夜用，惡丹皮殺鐵消金。 磁石必須先用醋酸泡製，把Fe3+改成Fe2+ 才好吸收

● 磁石，含鐵劑故可補血(養血安神)(吸納上下，維繫陰陽)

● 腎不納氣之喘：蛤蚧 補骨脂 沉香 磁石 紫河車


【相關藥對】:

    ● 磁石+石菖蒲 : 1.腎水不足，虛火上炎，以致耳鳴、耳聾等症；2.陰虛陽亢，以致頭暈頭痛、心悸心煩、失眠等症。

    ● 石決明+磁石 : 1.肝腎陰虛，水不涵木，以致肝陽上擾，症見頭暈、目眩、頭脹、頭痛、耳鳴、耳聾、失眠多夢、頭重腳輕等症；2.高血壓病。

    ● 紫石英+磁石 : 1.腎陰不足，水不涵木，肝陽上逆，以致頭昏、耳鳴、失眠、多夢等症；2.高血壓病，症屬虛性者。

    ● 珍珠母+磁石+朱砂+神曲 : 1.肝腎不足，肝陽上逆，以致頭暈、眼花、瞳孔散大、視物不明，以及耳鳴、耳聾等症；2.高血壓病，症屬虛性者；3.青光眼。

龍骨
=========================


龍骨
性味甘、澀，平
歸經心肝腎
主治鎮驚安神，平肝潛陽，收斂固澀
運用1、用於心神不寧，心悸失眠，驚癇癲狂2、用於肝陽眩暈3、用於滑脫諸證4、用於濕瘡癢疹、瘡瘍久潰不愈
藥性      




● 龍骨含有鎂和鈣質，鎂離子、鈣離子可以使腦神經休息，藥理作用是鎮靜

● 龍骨: 鈣與鎂, 作用在上面, 在心, 以鎮靜為主. 飛龍在天。   牡礪: 鈣與鋅, 作用在下面, 在腎, 潛藏在地。 龍骨 牡礪合用則心腎相交。

● 大人的遺精，可等同就是小孩子的遺尿，所以治療小孩子尿床，大多從大人的遺精方向著眼:   
  • 第一組藥是入腦、通腦竅的藥:  遠志、菖蒲
  • 第二組藥就是用一些鎮靜的藥: 龍骨、牡蠣。
  • 第三組藥是用一些安神的藥: 柏子仁、遠志
  • 最後第四組藥用收澀劑: 蓮蕊鬚、五味子、山茱萸、益智仁、覆盆子


● 滑脫諸證：龍骨 牡蠣

水龍骨科植物:  石韋+骨碎補


【相關藥對】:

    ● 龍骨+牡蠣 : 治療：1.心神不寧，心悸失眠，驚癇癲狂。

    ● 山茱萸+龍骨+牡蠣 : 虛汗淋漓，或喘逆，或怔忡，或氣虛不足以息。

    ● 山茱萸+人參+麥冬+龍骨+牡蠣 : 虛汗淋漓，或喘逆，或怔忡，或氣虛不足以息。

    ● 山茱萸+人參+龍骨+牡蠣 : 虛汗淋漓，或喘逆，或怔忡，或氣虛不足以息。

    ● 桂枝+龍骨+牡蠣 : 治療氣上衝，失眠，肝陽上亢。

    ● 龍骨+牡蠣+烏梅 : 治療上焦鬱熱，上熱下寒證。心悸失眠，虛熱消渴，容易上火。

    ● 龍骨+茜草+陽起石 : 卵巢癌、子宮癌、子宮頸癌，症見陰道出血、頻尿等。

    ● 生半夏+龍骨+牡蠣 : 治療腦癌、失眠。

[herb]一一三、龍骨：

龍骨【本經原文】味甘平，主心腹鬼注，精物老魅，欬逆洩利膿血，女子漏下，癥瘕堅結，小兒熱氣驚癇齒。主小兒大人驚癇癲疾狂走，心下結氣，不能喘息，諸痙，殺精物，久服輕身，通神明延年。
龍骨【產地】多產於我國山西四川等處山岩中，有一種化石。
龍骨【性味】味甘，性平，無毒。
龍骨【主治】龍骨為固斂正氣，鎮納浮陽要藥，主安心神辟驚煩，治洩痢膿血，崩帶遺精，鎮喘逆，斂虛汗，收瘡口，療脫肛。
龍骨【別錄】心腹煩滿，恚怒，氣伏在心下不得喘息，腸癰內疽，陰蝕，四肢痿枯，夜臥自驚汗出，止汗，縮小便，溺血，養精神，定魂魄，安五臟，白龍骨主多寐洩驚，小便洩精。
龍骨【甄權】逐邪氣，安心神，止夜夢鬼交，虛而多夢紛紜，止冷痢，下膿血，女子崩中帶下。
龍骨【日華】懷孕漏胎，止腸風下血，鼻洪吐血，止瀉痢渴疾，健脾，澀腸胃。
龍骨【藥徵】主治臍下動也，旁治煩驚失精。
龍骨【靈胎】斂氣澀血，定志安神。
龍骨【用量】普通二錢至三四錢。
龍骨【禁忌】收濇太過，若非久痢虛脫者，切勿妄投，火盛失精者誤用，多致溺赤濇痛，得人參牛黃良，畏石膏川椒，忌魚及鐵器。
龍骨【備註】龍骨今所用者皆石質也，過去有人誤解為龍之骨化石，實則為化石之一種，質細而性純，如同禹餘糧亦為石之一種，故二者功能相近，常與牡蠣合用。係土中石類，潛藏於土中，是天水之陽，以歸地下，故能潛納腎氣：收斂心神。因其潛納之性也。
龍骨【炮製】揀去雜質，刷去泥沙即得，煅龍骨即置鐵鍋中，煅透，即得。
