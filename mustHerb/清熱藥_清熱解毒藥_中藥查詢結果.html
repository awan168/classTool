金銀花
=========================


金銀花
性味甘，寒
歸經肺心胃
主治清熱解毒，疏散風熱
運用1、瘡癰疔腫2、外感風熱，溫病初起3、熱毒血痢
藥性     




● 抗過敏藥(皮膚病常用): 連翹，蟬蛻，殭蠶，防風，地龍，金銀花，元參，桑白皮

● 金銀花、地骨皮用水浸後都可以用來止癢

● 甘草、金銀花，對於緩解體內毒素和病毒入侵的作用很明顯

● 去青春痘: 用赤小豆、薏仁、黃芩、連翹、金銀花 

● 連翹會抑制胃酸，金銀花可以增加胃酸，二者合用相輔相成來治療皮膚的紅腫熱痛，是很好的消炎藥

● 治療一切癰腫疔瘡陽證之要藥：金銀花

忍冬科植物: 金銀花+附藥忍冬籐


【相關藥對】:

    ● 金銀花+連翹 : 治療外感風熱，溫病初起，咽喉疼痛，癰腫疔毒瘡瘍。

    ● 板藍根+金銀花 : 治療風熱證的皮膚病，紅疹，德國麻疹。

    ● 石膏+金銀花+連翹 : 治療發熱，咽喉疼痛，舌質紅，苔薄白，脈浮數。

    ● 金銀花+連翹+蘆根+大青葉+板藍根+黃芩 : 治療流行性乙型腦炎。

    ● 金銀花+土茯苓 : 治療梅毒的要藥。

    ● 金銀花+忍冬藤 : 1.溫病初起，邪在衛分者，或外感風熱，以致發熱惡風、咽喉腫痛、四肢酸楚疼痛等症；2.瘡瘍紅腫諸症；3.熱痹（類似結節性紅斑，風濕性關節炎有風濕活動者）諸症；4.脈管炎諸症。

    ● 金銀花+蒲公英 : 乳癌、乳癰，乳房局部紅腫熱痛或乳房患處發熱。

連翹
=========================


連翹
性味苦，微寒
歸經肺心膽
主治清熱解毒，消癰散結，疏散風熱
運用1、瘡癰腫毒，瘰癧結核2、外感風熱，溫病初起
藥性     




● 抗過敏藥(皮膚病常用): 連翹，蟬蛻，殭蠶，防風，地龍，金銀花，元參，桑白皮

● 天然抗生素、消炎藥:黃連、黃芩、黃柏、連翹											       

● 【倪師】不管是甲狀腺腫瘤、鼻咽癌、淋巴腺腫瘤、扁桃腺腫瘤，只要是喉嚨這一段有硬塊，只有兩味藥在用，就是桔梗和連翹(甚者加巴豆)

● 連翹:外感風熱 + 咽痛

● 去青春痘: 用赤小豆、薏仁、黃芩、連翹、金銀花 

● 連翹是內服的皮膚消炎藥，可是過了兩錢就傷胃

● 連翹會抑制胃酸，金銀花可以增加胃酸，二者合用相輔相成來治療皮膚的紅腫熱痛，是很好的消炎藥

● 化膿藥: 桔梗、枳實、連翹

● 諸痛癢瘡皆屬於心火，藥用連翹

● 保護血管、強化微血管、治療熱迫血妄行: 白茅根、藕節、連翹

● 頭皮屑、頭皮癢、頭皮出油的水煮洗液: 麻杏甘石湯＋蒼耳子、苦參根、苦參、黃柏、百部、連翹。(這幾味藥有抑制細菌病毒成長的效果)

● 病毒或濕熱引起的陰道炎: 黃柏、百部、苦參子、蛇床子、連翹 (除臭的部分:土茯苓、百部)

● 「瘡家聖藥」: 連翹

● 清心利尿： 連翹 木通 瞿麥 竹葉 淡竹葉

木犀科植物: 秦皮+連翹+女貞子


比較同異
梔子除煩煩悶而胸中窒
黃連煩悸而心下痞
連翹煩汗而咽中痛



【相關藥對】:

    ● 金銀花+連翹 : 治療外感風熱，溫病初起，咽喉疼痛，癰腫疔毒瘡瘍。

    ● 黃柏+梔子+連翹+薏苡仁 : 治療濕熱證的皮膚病。紅疹，皮膚癢，皮膚發紅，皮膚發熱，皮膚痛，舌質紅苔黃膩，脈數。

    ● 石膏+金銀花+連翹 : 治療發熱，咽喉疼痛，舌質紅，苔薄白，脈浮數。

    ● 柴胡+黃芩+連翹 : 治療發熱，或反覆發燒。

    ● 連翹+梔子 : 治療瘡癰腫毒，瘰癧結核。

    ● 連翹+梔子+黃柏 : 治療小便赤，腳濕氣(腳癬)，濕疹，女性盆腔炎，陰道炎。

    ● 金銀花+連翹+蘆根+大青葉+板藍根+黃芩 : 治療流行性乙型腦炎。

    ● 連翹+薏苡仁 : 治療各種皮膚病，皮膚癢，瘡癰，急性化膿。

    ● 連翹+梔子+黃芩 : 治療瘡癰腫毒，瘰癧結核。

    ● 蔓荊子+連翹 : 1.風熱聚於上焦，以致頭昏、頭痛、發熱等症；2.風火頭痛，暴發火眼等症。

    ● 牛蒡子+連翹 : 1.熱聚上焦，以致口舌生瘡、牙齦腫痛、咽喉腫痛等症；2.癰腫瘡瘍諸症；3.風熱癢疹、斑疹等症；4.面癱（面神經麻痹）初期；證屬病毒感染，耳後乳突，翳風穴周圍疼痛者宜用。

    ● 連翹 : 1、瘡癰腫毒，瘰癧結核2、外感風熱，溫病初起

大青葉
=========================


大青葉
性味苦，大寒
歸經心肺胃
主治清熱解毒，涼血消斑
運用1、瘡癰丹毒，口瘡，咽痛2、外感風熱，溫病初起3、熱入營血，高熱斑疹
藥性 




● 十字花科的菘藍，其根叫做板藍根，它的葉子叫做大青葉。大青葉濃縮成的粉劑叫青黛

● 大青葉有黃酮素可使黏傷口癒合 (蜂膠也有黃酮素可癒合傷口)

● 治血熱毒盛所致諸證之要藥：大青葉

十字花科植物: 大青葉+板藍根+薺菜+萊菔子+白芥子+葶藶子

[top]   [herb]三○二、連翹：

連翹【本經原文】一名異翹，一名蘭華，一名折根，一名軹，一名三廉。味苦，平，無毒。治寒熱，鼠瘻，瘰癧，癰腫，惡瘡，癭瘤，結熱，蠱毒。生山谷。
連翹【產地】產山谷中及澤地，處處有之，四川產者為勝。
連翹【性味】味苦辛，性平，無毒。
連翹【主治】連翹為散結清火要藥，主治瘡瘍，能清火解毒，消腫排膿，散血結氣聚，去上焦熱，通利小便。
連翹【別錄】去白虫。
連翹【甄權】通利五淋，小便不通，除心家客熱。
連翹【大明】通小腸，排膿治瘡癤，止痛，通月經。
連翹【丹溪】瀉心火，除脾胃溼熱，治中部血證以為使。
連翹【東垣】散諸經血結氣聚，消腫。
連翹【容川】連翹亦象心包，而質輕揚，味微苦，故輕揚上達，能清心與上焦頭目之熱，其無辛散之氣，故不主散。
連翹【靈胎】散鬱火，治瘡瘍。
連翹【倪註】此藥常與銀花同用，皆屬味清而質輕，皆能升清氣。
連翹【用量】普通一錢至三錢。
連翹【禁忌】連翹清而無補之藥也，癰疽已潰及日久，火熱由於虛，與脾胃薄弱作泄者均忌，忌火。

[top]   [herb]二九八、白頭翁：

白頭翁【本經原文】一名野丈人，一名胡王使者。味苦，溫，有毒。治溫瘧，狂易，寒熱，癥瘕積聚，癭氣，逐血，止痛，治金瘡。生山谷及曰野。
白頭翁【產地】處處有之，生山谷及田野，近根處有白茸，故名。
白頭翁【性味】味辛苦，性溫，無毒，【或作小毒】。
白頭翁【主治】白頭翁為治痢要藥 功能洩熱涼血，治毒痢赤痢，下血腹痛 及婦女經閉。
白頭翁【別錄】治鼻衄。
白頭翁【甄權】赤痢腹痛，齒痛，百節骨痛，項下癭瘤。
白頭翁【弘景】止毒痢。
白頭翁【容川】白頭翁一莖直上如柴胡，所以能治下痢後重者，升散鬱結也，其無風動搖，有風不動，色白有毛，凡毛皆得風氣，採於秋月，得金木交合之氣，故能熄風，從肺金以達風木之氣，使木不侮土，故功在升舉後重，而止痢疾，又其味苦，是治風熱之妙藥，仲景治產後中風，及痢疾後重者，是取其熄風火達肝陽也。
白頭翁【藥徵】主治熱利下重也。
白頭翁【用量】普通錢半至三錢。
白頭翁【禁忌】滯下胃虛不思食，及完穀不化，泄瀉由虛寒，寒熱而不由濕毒者，均忌，得酒良。

[top]   [herb]二七八、射干：

射干【本經原文】味苦，平，有毒。治欬逆上氣，喉痺，咽痛，不得消息，散結氣，腹中邪逆，食飲大熱。生川谷田野。一名烏扇，一名烏蒲。
射干【產地】處處有之，生於各地山中，為多年生草本，園圃亦可栽種，葉似菖蒲，其花可供玩賞。
射干【性味】味苦，性平，有毒。
射干【主治】射干清火解毒散血消痰要藥，主咽喉腫痛，欬逆上氣，降實火，利大腸，治積痰瘀血，熱氣結毒。
射干【別錄】療老血在心脾間，欬唾言語氣臭，散胸中熱氣。
射干【甄權】治疰氣，消瘀血，通女人月閉。
射干【大明】消痰破癥結，胸膈滿腹脹，氣喘，痃癖，開胃下食，鎮肝明目。
射干【元素】去胃中癰瘡。
射干【用童】普通錢半至三錢。
射干【禁忌】性不益陰，凡脾胃弱，藏寒，氣血虛，病無實熱，均忌。
射干【容川】射干微苦，利喉中之痰。
