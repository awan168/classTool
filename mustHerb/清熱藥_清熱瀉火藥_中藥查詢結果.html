石膏
=========================


石膏
性味辛、甘，大寒
歸經肺胃
主治清熱瀉火，除煩止渴收斂生肌
運用1、用於氣分實熱證 2、用於肺熱咳喘 3、用於胃火牙痛 
藥性     




● 石膏是中風傷寒的用藥，得麻、桂而助青龍(麻黃)之力; 得知母、甘草有白虎之號 

● 石膏色白且味甘辛，是為陽藥

● 石膏與麻黃同用就不會大汗出而衰竭

● 清六腑之熱: 石膏、滑石、寒水石

● 知母滑腸，石膏補水分補太多會傷到腸胃，故用甘草及粳米來保護腸胃(見白虎湯)

● 麻黃配桂枝則發汗，配石膏則行水，配射干則定喘

● 【倪師】吃一錢甘遂，要吃十斤石膏還有人參，津液才補得回來。

● 治氣分實熱和肺胃實火之要藥：石膏

● 胃火牙痛：石膏


 
比較同異
 
黃耆收汗汗出而腫，不煩
 
石膏汗出而渴，煩
 
 


【相關藥對】:

    ● 麻黃+石膏 : 治療邪熱壅肺的咳喘，全身水腫。

    ● 石膏+知母 : 治療氣分實熱，心煩、身熱、喜冷飲。

    ● 石膏+生地黃 : 治療陰虛熱擾，虛煩少寐。

    ● 黃柏+石膏 : 治療湯火燙傷及濕疹。

    ● 桔梗+石膏 : 治療咽喉痛，乾咳無痰或黃稠痰。

    ● 石膏+金銀花+連翹 : 治療發熱，咽喉疼痛，舌質紅，苔薄白，脈浮數。

    ● 桔梗+石膏+知母 : 治療發熱，口渴，咽喉痛，舌紅，脈洪大。

    ● 石膏+魚腥草+黃芩 : 治療感冒肺炎、肺膿瘍。咳嗽，痰黃稠，黃鼻涕，舌紅苔黃，脈數。

    ● 桔梗+石膏+薑半夏 : 治療咳嗽，不易咳出的濃痰，黏痰。

    ● 石膏+柴胡+黃芩 : 治療感冒發高燒，反覆發燒。舌質紅，脈浮弦數。

    ● 桔梗+石膏+天花粉+葛根 : 治療口乾、咽乾，舌質紅，脈數。

    ● 薄荷+石膏 : 外感病，證屬風熱為患，以致惡寒輕，或惡風，發熱重，或晝涼暮熱，久久不退等症。

    ● 羌活+石膏 : 1.流行性感冒，上呼吸道感染，證屬風寒閉遏肌表，裡有蘊熱者；2.暑溫、暑濕，證屬風寒外束，肌表受累，內有鬱熱者；3.乙型腦炎，邪在衛分，或衛氣同病者。

    ● 石膏+細辛 : 1.內蘊鬱熱，隨經上竄，以致牙痛、牙齦腫痛、口舌生瘡等症；2.感受風熱，上竄清竅，以致頭痛諸症。

    ● 淡竹葉+石膏 : 1.口舌生瘡，牙齦腫痛，證屬心胃熱盛者；2.溫熱病後期，餘邪未清，低熱不退，胸中煩悶，舌紅、少苔等症；3.糖尿病之煩熱咳逆、乾渴多飲等症。

    ● 石膏 : 1、用於氣分實熱證

    ● 炮附子+石膏 : 全身痛證兼有局部發熱。

知母
=========================


知母
性味苦、甘，寒
歸經肺胃腎
主治清熱瀉火，滋陰潤燥
運用1、用於氣分實熱證2、用於肺熱咳嗽，陰虛燥咳3、用於陰虛消渴4、用於骨蒸潮熱
藥性     




● 石膏是中風傷寒的用藥，得麻、桂而助青龍(麻黃)之力; 得知母、甘草有白虎之號 

● 知母、黃柏是苦寒藥，久服會傷脾胃，女性久服則絕孕

● 知母清太陰之氣

● 清涼為肺之母也，知母者，知肺之母以清涼也。

● 知母滑腸，石膏補水分補太多會傷到腸胃，故用甘草及粳米來保護腸胃(見白虎湯)

● 知母潤肺滋腎而降火

● 退虛熱：知母 黃柏 牡丹皮 青蒿 白薇 地骨皮 銀柴胡 胡黃連 秦艽 龜甲 鱉甲

● 潤腸通便：火麻仁 郁李仁 柏子仁 核桃仁 桃仁  決明子 榧子 蘇子 冬葵子 栝樓 當歸 何首烏 黑芝麻 桑葚 肉蓯蓉 胖大海 知母 生地黃 鎖陽 杏仁

百合科植物: 蔥白+知母+重樓+土茯苓+蘆薈+薤白+川貝母+浙貝母+韭菜子+百合+麥冬+天冬+玉竹+黃精+大蒜


【相關藥對】:

    ● 石膏+知母 : 治療氣分實熱，心煩、身熱、喜冷飲。

    ● 知母+黃柏 : 治療陰虛潮熱，骨蒸盜汗。

    ● 川貝母+知母 : 治療肺熱虛實咳嗽。

    ● 桔梗+石膏+知母 : 治療發熱，口渴，咽喉痛，舌紅，脈洪大。

    ● 知母+草果 : 1.表裡不和，乍寒乍熱，寒熱往來等症；2.瘧疾（包括瘴瘧）諸症。

    ● 知母+黃柏+紫油肉桂 : 1.糖尿病，表現為“腎消”，也叫“下消”者，症見多尿，小便混濁、如膏如脂等，2.糖尿病兼見下身瘙癢者。

    ● 百合+知母 : 1.陰虛或溫熱病後餘熱未清，以致頭昏、心煩不安、失眠，證屬心中熱鬱氣悅者；2.情志不遂，以致精神恍惚、不能自制等症；3.百合病。

    ● 黃耆+知母 : 1.惡性腫瘤，術後、放療、化療之後，證屬氣陰兩虛，大氣下陷者；2.胃脘痛，證屬氣陰兩虛者；3.原發性血小板減少性紫癜；4.肺結核、腎結核，證屬陰虛內熱者；5.慢性腎炎血尿，證屬氣陰兩虛者。

    ● 知母+黃柏+玄參 : 陰虛有火之證。症見發熱盜汗、口乾咽燥、消瘦、大便乾燥，舌質紅、脈細數。

梔子
=========================


梔子
性味苦，寒
歸經心肝肺胃三焦
主治瀉火除煩，清熱利濕，涼血解毒
運用1、用於熱病煩悶2、用於濕熱黃疸3、用於血熱出血4、用於熱毒瘡瘍
藥性     




● 梔子之性能收善降，可止胃酸反逆

● 梔子通瀉三焦火毒，使諸火毒下行從膀胱出

● 梔子是治胸中懊憹的特效藥

● 一般都說:黃芩瀉上焦火，黃連瀉中焦火，黃柏瀉下焦火 。而梔子通瀉三焦之火 (生大黃瀉上焦頭目的熱)				       

● 五鬱之法: 香附開氣鬱，蒼朮除濕鬱，川芎行血鬱，梔子清火鬱，神麴消食鬱

● 梔子是很好的消炎藥，因為是茜草科植物，也有利於活血化瘀。 

茜草科植物: 梔子+白花蛇舌草+雞矢籐+茜草+鉤籐+巴戟天


比較同異

連翹煩汗而咽中痛

[top]   [herb]一四七、石膏：

石膏【本經原文】味辛微寒，主中風寒熱，心下逆氣，驚喘口乾，舌焦不能息，腹中堅痛，除邪鬼，產乳金創。又名寒水石。
石膏【產地】產浙江江蘇山東，他省亦有出產。
石膏【性味】味甘辛，性徵寒，無毒。
石膏【主治】石膏為清涼解熱要藥，主治肺胃三焦氣分之熱，大渴熱狂，能解肌發汗，又能清熱止汗。
石膏【別錄】除時氣頭痛身熱，三焦大熱，皮膚熱，腸胃中結氣，解肌發汗，止消渴煩逆腹脹，暴氣喘咽熱，亦可作浴湯。
石膏【甄權】治傷寒頭痛如裂，壯熱皮如火燥，和蔥煎茶去頭痛。
石膏【大明】治天行熱狂頭風旋，下乳，揭齒益齒。
石膏【元素】止陽明經頭痛，發熱惡寒；日哺潮熱，大渴引飲，中暑潮熱，牙痛。
石膏【容川】得石之性多，味雖鹹而不甚，且此山之石即能生水，流而為泉，是故此石純具水性，稟天水之寒，故能清熱。
石膏【藥徵】主治煩渴，旁治譫語煩躁身熱。
石膏【用量】普通三錢至五錢，大劑二三兩。
石膏【禁忌】傷寒中風太陽症，少陽症之往來寒熱等均忌，真寒假熱之症或因裡虛引起之虛熱症亦忌，有金瘡下乳亦忌，胃家寒症者亦忌，惡巴豆畏鐵。
