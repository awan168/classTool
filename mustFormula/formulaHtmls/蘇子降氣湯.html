【蘇子降氣湯】

出處太平惠民和劑局方
功用降氣平喘，祛痰止咳。
應用喘咳短氣，痰涎壅盛，胸膈滿悶，上盛下虛；或腰疼腳弱肢體怠倦，或肢體浮腫，舌苔白滑或白膩。
組成● 蘇子 : 降氣化痰，止咳平喘，潤腸通便
● 生半夏 : 燥濕化痰，降逆止嘔，消痞散結，外用消腫止痛
● 當歸 : 補血，活血，調經，止痛，潤腸
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 前胡 : 降氣化痰，宣散風熱
● 厚朴 : 燥濕，行氣，消積，平喘
● 肉桂 : 補火助陽，散寒止痛，溫經通脈
● 陳皮 : 理氣健脾，燥濕化痰
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 大棗 : 補中益氣，養血安神，緩和藥性
● 紫蘇葉 : 發汗解表，行氣寬中，解魚蟹毒
