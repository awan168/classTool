治療中風的方劑

【疏經活血湯】

出處金匱要略
功用疏經活血，祛風除濕。
應用遍身走痛如刺，左足痛尤甚，晝輕夜重，以及婦人血風勞。可應用在筋肉痛、風濕病、痛風、漿液性膝關節炎、腰痛、坐骨神經痛、高血壓、下肢麻痺、腳氣、浮腫、紫斑病、靜脈曲張、半身不遂，腦卒中後遺症、產後血栓性疼痛等。
組成● 當歸 : 補血，活血，調經，止痛，潤腸
● 生地黃 : 清熱涼血，養陰生津
● 蒼朮 : 燥濕健脾，祛風濕，發表
● 川芎 : 活血行氣，祛風止痛
● 桃仁 : 活血祛瘀，潤腸通便，止咳平喘
● 茯苓 : 利水滲濕，健脾安神
● 酒白芍 : 養血調經，平肝止痛，斂陰止汗
● 川牛膝 : 活血通經，補肝腎，強筋骨，引火（血）下行，利尿通淋
● 威靈仙 : 祛風濕，通經絡，消痰水，治骨鯁
● 防己 : 祛風濕，止痛，利水消腫
● 羌活 : 發散風寒，勝濕止痛
● 防風 : 祛風解表，勝濕止痛止痙
● 龍膽草 : 清熱燥濕，瀉肝火
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 陳皮 : 理氣健脾，燥濕化痰
● 白芷 : 祛風散寒，通竅止痛，消腫排膿，燥濕止帶
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
