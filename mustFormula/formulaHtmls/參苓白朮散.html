【參苓白朮散】

出處金匱要略
功用益氣健脾，滲濕止瀉。
應用脾胃氣虛夾濕證。四肢乏力，形體消瘦、飲食不化或吐或瀉胸脘悶脹，面色萎黃、舌淡苔白膩，脈虛緩。
組成● 白扁豆 : 健脾化濕，和中消暑，解毒
● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 白朮 : 補氣健脾，燥濕利水，固表止汗，安胎
● 茯苓 : 利水滲濕，健脾安神
● 炙甘草 : 補脾和胃，益氣複脈。
● 山藥 : 益氣養陰，補脾肺腎，固精止遺
● 蓮子 : 補脾止瀉，固澀止帶，益腎固精，養心安神
● 薏苡仁 : 利水滲濕，健脾止瀉，清熱排膿，除痹
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 砂仁 : 化濕開胃，溫脾止瀉，理氣安胎
● 大棗 : 補中益氣，養血安神，緩和藥性
