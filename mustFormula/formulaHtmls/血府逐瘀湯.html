【血府逐瘀湯】

出處太平惠民和劑局方
功用活血化瘀，行氣止痛。
應用胸中血瘀、血行不暢所致之胸痛、頭痛日久不癒、痛如針刺而有定處，或呃逆日久不止，或內熱煩悶、心悸失眠、急躁善怒、入暮漸熱，舌質黯紅、舌邊有瘀斑或舌面有瘀點、唇暗或兩目暗黑、脈澀或弦緊。
組成● 桃仁 : 活血祛瘀，潤腸通便，止咳平喘
● 紅花 : 活血通經，祛瘀止痛
● 當歸 : 補血，活血，調經，止痛，潤腸
● 生地黃 : 清熱涼血，養陰生津
● 川芎 : 活血行氣，祛風止痛
● 赤芍 : 清熱涼血，祛瘀止痛
● 川牛膝 : 活血通經，補肝腎，強筋骨，引火（血）下行，利尿通淋
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 柴胡 : 疏散退熱，疏肝解郁，升舉陽氣，清膽截瘧
● 枳殼 : 破氣除痞，化痰稍積，寬腸胃，長于行氣寬中除脹
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
