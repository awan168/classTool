【洗肝明目湯】

出處華岡青洲驗方
功用清熱袪風，消腫明目。
應用風熱引起之目睛紅赤、腫脹、羞明流淚、目澀疼痛，舌紅苔薄黃，脈數。
組成● 當歸 : 補血，活血，調經，止痛，潤腸
● 赤芍 : 清熱涼血，祛瘀止痛
● 黃芩 : 清熱燥濕，瀉火解毒，止血，安胎
● 連翹 : 清熱解毒，消癰散結，疏散風熱
● 川芎 : 活血行氣，祛風止痛
● 生地黃 : 清熱涼血，養陰生津
● 梔子 : 瀉火除煩，清熱利濕，涼血解毒
● 防風 : 祛風解表，勝濕止痛止痙
● 荊芥 : 祛風解表，透疹止癢止血
● 羌活 : 發散風寒，勝濕止痛
● 決明子 : 清肝明目，潤腸通便
● 薄荷 : 發散風熱，清利咽喉，透疹解毒，疏肝解郁
● 蔓荊子 : 發散風熱，清利頭目
● 菊花 : 發散風熱，清肝明目，平抑肝陽，清熱解毒
● 黃連 : 清熱燥濕，瀉火解毒
● 蒺藜 : 平肝疏肝，祛風明目
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 石膏 : 清熱瀉火，除煩止渴收斂生肌
