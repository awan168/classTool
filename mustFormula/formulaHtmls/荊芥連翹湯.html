【荊芥連翹湯】

出處金匱要略
功用疏風清熱、消腫止痛。
應用肝腎經風熱、兩耳腫痛、或膽熱移腦、致患鼻淵、舌偏紅、苔薄黃、脈數或浮數者。
組成● 荊芥 : 祛風解表，透疹止癢止血
● 連翹 : 清熱解毒，消癰散結，疏散風熱
● 防風 : 祛風解表，勝濕止痛止痙
● 柴胡 : 疏散退熱，疏肝解郁，升舉陽氣，清膽截瘧
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 白芷 : 祛風散寒，通竅止痛，消腫排膿，燥濕止帶
● 枳殼 : 破氣除痞，化痰稍積，寬腸胃，長于行氣寬中除脹
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 薄荷 : 發散風熱，清利咽喉，透疹解毒，疏肝解郁
● 黃連 : 清熱燥濕，瀉火解毒
● 黃芩 : 清熱燥濕，瀉火解毒，止血，安胎
● 黃柏 : 清熱燥濕，瀉火解毒，瀉火解毒
● 梔子 : 瀉火除煩，清熱利濕，涼血解毒
● 生地黃 : 清熱涼血，養陰生津
● 當歸 : 補血，活血，調經，止痛，潤腸
● 川芎 : 活血行氣，祛風止痛
● 白芍 : 養血調經，平肝止痛，斂陰止汗
