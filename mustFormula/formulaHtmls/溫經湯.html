【溫經湯】

出處金匱要略
功用溫經散寒，養血祛瘀。
應用衝任虛寒，瘀血阻滯證。漏下不止，月經不調，或前或後，或逾期不止，或一月再行，或經停不至，而見傍晚發熱，手心煩熱，少腹裡急，腹滿。亦治婦人宮冷，久不受孕。
組成● 吳茱萸 : 散寒止痛，疏肝降逆，助陽止瀉
● 當歸 : 補血，活血，調經，止痛，潤腸
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 川芎 : 活血行氣，祛風止痛
● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 桂枝 : 發汗解肌，溫經通脈，通陽化氣
● 黃明膠 : 補血，止血，滋陰潤燥
● 牡丹皮 : 清熱涼血，活血散瘀
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 炙甘草 : 補脾和胃，益氣複脈。
● 生半夏 : 燥濕化痰，降逆止嘔，消痞散結，外用消腫止痛
● 麥門冬 : 養陰潤肺，益胃生津，清心除煩
