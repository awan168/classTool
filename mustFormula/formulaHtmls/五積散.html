【五積散】

出處仙授理傷續斷秘方
功用解表散寒，溫中消積。
應用1.外感風寒、內傷生冷之證、身熱無汗、頭痛身疼、項背腰股拘急，胸滿惡食、嘔吐腹痛。 2.婦女心腹疼痛，月經不調等屬寒凝氣滯者。
組成● 茯苓 : 利水滲濕，健脾安神
● 蒼朮 : 燥濕健脾，祛風濕，發表
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 枳殼 : 破氣除痞，化痰稍積，寬腸胃，長于行氣寬中除脹
● 陳皮 : 理氣健脾，燥濕化痰
● 麻黃 : 發汗解表，宣肺平喘，利水消腫
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 白芷 : 祛風散寒，通竅止痛，消腫排膿，燥濕止帶
● 川芎 : 活血行氣，祛風止痛
● 當歸 : 補血，活血，調經，止痛，潤腸
● 炙甘草 : 補脾和胃，益氣複脈。
● 肉桂 : 補火助陽，散寒止痛，溫經通脈
● 生半夏 : 燥濕化痰，降逆止嘔，消痞散結，外用消腫止痛
● 厚朴 : 燥濕，行氣，消積，平喘
● 乾薑 : 溫中散寒，回陽通脈，溫肺化飲
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
