【清上防風湯】

出處萬病回春
功用清熱解毒，袪風除濕。
應用以上焦的實熱爲適用本方的目標，用於上部（顔面或頭部)有血熱鬱滯、發瘡、顔面紅赤而上衝者。集中於上部的熱邪，以在上部發表或清解爲宜。體質不甚虛弱，面疱等亦多呈現紅紫色。 本方能發散清解鬱滯於上焦部尤其是顔面的熱，可應用於下列的各種疾患：面疱（以强壯的年輕男子爲多，年輕女子而身體壯實，臉色紅者也會發生）的發疹充血而紅赤者、頭部濕疹、眼充血、顏面充血、酒渣鼻等症。
組成● 川芎 : 活血行氣，祛風止痛
● 黃芩 : 清熱燥濕，瀉火解毒，止血，安胎
● 連翹 : 清熱解毒，消癰散結，疏散風熱
● 防風 : 祛風解表，勝濕止痛止痙
● 白芷 : 祛風散寒，通竅止痛，消腫排膿，燥濕止帶
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 梔子 : 瀉火除煩，清熱利濕，涼血解毒
● 荊芥 : 祛風解表，透疹止癢止血
● 黃連 : 清熱燥濕，瀉火解毒
● 枳殼 : 破氣除痞，化痰稍積，寬腸胃，長于行氣寬中除脹
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 薄荷 : 發散風熱，清利咽喉，透疹解毒，疏肝解郁
