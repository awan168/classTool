【托裡消毒散】

出處萬病回春
功用托毒潰膿，補氣養血。
應用癰疽氣血俱虛，腫不能潰，或潰不能歛。
組成● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 金銀花 : 清熱解毒，疏散風熱
● 川芎 : 活血行氣，祛風止痛
● 茯苓 : 利水滲濕，健脾安神
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 白芷 : 祛風散寒，通竅止痛，消腫排膿，燥濕止帶
● 黃耆 : 補氣升陽，益衛固表，利水消腫，托瘡生肌
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 當歸 : 補血，活血，調經，止痛，潤腸
● 皂角刺 : 活血消腫，排膿通乳
● 白朮 : 補氣健脾，燥濕利水，固表止汗，安胎
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 連翹 : 清熱解毒，消癰散結，疏散風熱
