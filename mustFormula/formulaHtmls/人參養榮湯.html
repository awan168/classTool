【人參養榮湯】

出處三因極一病證方論
功用益氣補血，養心安神。
應用氣血兩虛、積勞虛損、呼吸少氣、形瘦神疲、面色萎黃、心虛驚悸、食少作瀉、咽乾唇燥、婦女月經不調、舌淡、脈細弱。
組成● 黃耆 : 補氣升陽，益衛固表，利水消腫，托瘡生肌
● 當歸 : 補血，活血，調經，止痛，潤腸
● 桂心 : 引血化汗化膿，內託癰疽痘瘡，益精明目，止瘀生肌。補勞傷，暖腰膝，續筋骨。
● 炙甘草 : 補脾和胃，益氣複脈。
● 陳皮 : 理氣健脾，燥濕化痰
● 白朮 : 補氣健脾，燥濕利水，固表止汗，安胎
● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 熟地黃 : 補血滋陰，益精填髓
● 五味子 : 斂肺滋腎，生津斂汗，澀精止瀉，寧心安神
● 茯苓 : 利水滲濕，健脾安神
● 遠志 : 寧心安神，祛痰開竅，消散癰腫
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 大棗 : 補中益氣，養血安神，緩和藥性
