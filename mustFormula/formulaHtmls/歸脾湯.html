【歸脾湯】

出處濟生方
功用益氣補血，健脾養心。
應用1.心脾氣血兩虛證。心悸怔忡健忘失眠、盜汗虛熱，食少体倦、面色萎黃，舌淡、苔白薄，脈細緩。 2.脾不統血證。便血、皮下紫癜，婦女崩漏，月經超前，量多色淡或淋漓不止或帶下。
組成● 白朮 : 補氣健脾，燥濕利水，固表止汗，安胎
● 當歸 : 補血，活血，調經，止痛，潤腸
● 茯苓 : 利水滲濕，健脾安神
● 黃耆 : 補氣升陽，益衛固表，利水消腫，托瘡生肌
● 遠志 : 寧心安神，祛痰開竅，消散癰腫
● 龍眼肉 : 補益心脾，養血安神
● 炒酸棗仁 : 養心益肝，安神，斂汗
● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 木香 : 行氣，調中，止痛
● 炙甘草 : 補脾和胃，益氣複脈。
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 大棗 : 補中益氣，養血安神，緩和藥性
