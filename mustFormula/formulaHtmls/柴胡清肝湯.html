【柴胡清肝湯】

出處傷寒論
功用清肝瀉熱。
應用肝膽經及三焦之實熱證。症見胸脅不舒，寒熱往來，口苦咽乾、咽痛或耳鳴、耳痛、皮膚搔癢、或婦人帶下陰癢、舌紅、苔黃膩、脈弦數或浮數。
組成● 柴胡 : 疏散退熱，疏肝解郁，升舉陽氣，清膽截瘧
● 當歸 : 補血，活血，調經，止痛，潤腸
● 赤芍 : 清熱涼血，祛瘀止痛
● 川芎 : 活血行氣，祛風止痛
● 生地黃 : 清熱涼血，養陰生津
● 黃連 : 清熱燥濕，瀉火解毒
● 黃芩 : 清熱燥濕，瀉火解毒，止血，安胎
● 黃柏 : 清熱燥濕，瀉火解毒，瀉火解毒
● 梔子 : 瀉火除煩，清熱利濕，涼血解毒
● 連翹 : 清熱解毒，消癰散結，疏散風熱
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 牛蒡子 : 發散風熱，宣肺透疹，利咽散結，解毒消腫
● 天花粉 : 清熱生津，消腫排膿
● 薄荷 : 發散風熱，清利咽喉，透疹解毒，疏肝解郁
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
