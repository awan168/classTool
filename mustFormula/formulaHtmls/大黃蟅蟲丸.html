【大黃蟅蟲丸】

出處金匱要略
功用祛瘀生新。
應用子宮肌瘤、卵巢囊腫、肝硬化、脾腫大、手術後腸粘連、子宮內膜性閉經、卵巢性閉經、血栓閉塞性脈管炎、腦血栓形成，症見腹滿疼痛拒按，痛處固定，納差，肌膚甲錯，兩目黯黑，脈弦澀者。
組成● 大黃 : 瀉下攻積，清熱瀉火，止血，解毒，活血祛瘀，清瀉濕熱
● 黃芩 : 清熱燥濕，瀉火解毒，止血，安胎
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 桃仁 : 活血祛瘀，潤腸通便，止咳平喘
● 杏仁 : 止咳平喘，潤腸通便
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 生地黃 : 清熱涼血，養陰生津
● 蜀漆 : 湧吐痰涎，截瘧
● 虻蟲 : 破血通經，逐瘀消癥
● 燙水蛭 : 破血逐瘀消癥
● 蠐螬 : 活血行氣
● 蟅蟲 : 逐瘀，破积，通络，理伤
