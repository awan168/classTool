【百合固金湯】

出處慎齋遺書
功用滋養肺腎，止咳化痰。
應用肺腎陰虧、虛火上炎證。咽喉乾燥、疼痛、咳嗽、氣喘、咳痰帶血、手足心熱、舌紅少苔、脈細數。
組成● 熟地黃 : 補血滋陰，益精填髓
● 生地黃 : 清熱涼血，養陰生津
● 當歸 : 補血，活血，調經，止痛，潤腸
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 桔梗 : 開宣肺氣，祛痰排膿，利咽
● 玄參 : 清熱涼血，滋陰解毒
● 川貝母 : 清化熱痰，潤肺止咳，散結消腫
● 麥門冬 : 養陰潤肺，益胃生津，清心除煩
● 百合 : 養陰潤肺止咳，清心安神
