【清燥救肺湯】

出處醫效秘傳
功用清燥潤肺，養陰益氣。
應用溫燥傷肺證。頭痛身熱，乾咳無痰，氣逆而喘，口乾而渴，鼻燥咽乾，心煩，胸滿脅痛，舌乾無苔或少苔，脈虛大而數。
組成● 當歸 : 補血，活血，調經，止痛，潤腸
● 白芍 : 養血調經，平肝止痛，斂陰止汗
● 茯苓 : 利水滲濕，健脾安神
● 白朮 : 補氣健脾，燥濕利水，固表止汗，安胎
● 柴胡 : 疏散退熱，疏肝解郁，升舉陽氣，清膽截瘧
● 牡丹皮 : 清熱涼血，活血散瘀
● 梔子 : 瀉火除煩，清熱利濕，涼血解毒
● 甘草 : 益氣補中，清熱解毒，祛痰止咳，緩急止痛，調和藥性
● 薄荷 : 發散風熱，清利咽喉，透疹解毒，疏肝解郁
● 生半夏 : 燥濕化痰，降逆止嘔，消痞散結，外用消腫止痛
● 厚朴 : 燥濕，行氣，消積，平喘
● 生薑 : 發汗解表、溫中止嘔，溫肺止咳
● 紫蘇葉 : 發汗解表，行氣寬中，解魚蟹毒
