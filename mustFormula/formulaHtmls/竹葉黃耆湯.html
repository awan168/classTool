【竹葉黃耆湯】

出處問止中醫大腦大數據分析組方
功用清熱生津，益氣養血。
應用消渴，氣血虛，胃火盛而作渴。以及癰疽、發背、諸瘡之大渴。
組成● 當歸 : 補血，活血，調經，止痛，潤腸
● 酒白芍 : 養血調經，平肝止痛，斂陰止汗
● 生地黃 : 清熱涼血，養陰生津
● 熟地黃 : 補血滋陰，益精填髓
● 陳皮 : 理氣健脾，燥濕化痰
● 小茴香 : 散寒止痛，理氣和中
● 補骨脂 : 補腎助陽，固精縮尿，暖脾止瀉，納氣平喘
● 懷牛膝 : 活血通經，補肝腎，強筋骨，引火（血）下行，利尿通淋
● 杜仲 : 補肝腎，強筋骨，安胎
● 茯苓 : 利水滲濕，健脾安神
● 人參 : 大補元氣，補脾益肺，生津止渴，安神益智
● 黃柏 : 清熱燥濕，瀉火解毒，瀉火解毒
● 知母 : 清熱瀉火，滋陰潤燥
● 炙甘草 : 補脾和胃，益氣複脈。
● 大棗 : 補中益氣，養血安神，緩和藥性
